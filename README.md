# helpbuttons backend


## run with npm
### install dependencies

```sh
npm install
```
### setup the datasources

```sh
npm run migrate
```

### Run the application
```sh
npm start
```

Open http://127.0.0.1:3001 in your browser.

### run the application in watchmode
```sh
npm start:watch
```
